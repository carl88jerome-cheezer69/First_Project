# First Project

My first HTML project — a simple personal profile page built while learning the basics of web development.

## About

This is a single-page site introducing myself, built with plain HTML as an introduction to web development fundamentals: page structure, semantic elements, and basic content organization.

## Sections

- **About** — a short introduction
- **Contact** — how to reach me
- **Interests** — web development and fitness
- **Hobbies** — gym, anime, video games, and family time

## Built With

- HTML5

## Getting Started

Clone the repo and open [index.html](index.html) in your browser — no build steps or dependencies required.

```bash
git clone <repo-url>
cd First_Project
```

Then open `index.html` directly, or serve it locally:

```bash
npx serve .
```
